# Server Configuration Guide

## Environment Variables for Port Configuration

The **SERVER_PORT** environment variable is used to configure the server port dynamically.

### Example Configuration:
- SERVER_PORT=8080 (for development)  
- SERVER_PORT=3000 (for production)

### CORS Configuration:
For web applications, proper CORS setup is required using CorsConfigurationSource bean.

### Health Check Endpoints:
- /health/ready - Readiness probe
- /health/live - Liveness probe
